## 1. 분석 배경 및 목적

### 왜 이 분석을 했는가?

최근 어린이, 노인 등 **교통약자(몸이 불편하거나 사고 위험에 더 취약한 사람들)** 의 교통사고가 꾸준히 증가하고 있습니다. 특히:
- 교차로 우회전 구간에서의 보행자 사고
- 스쿨존(어린이보호구역) 내 어린이 사고
- 야간·주말에 집중되는 고위험 도로 사고

이를 해결하기 위해, **단순히 사고 건수만 보는 것이 아니라** 사고의 심각성을 반영한 점수(EPDO)를 이용해 어느 도로/지역이 실제로 얼마나 위험한지 정밀하게 측정했습니다.

### 분석 대상 지역

| 구분 | 지역 |
|------|------|
| 분석 대상 (현황 파악) | 동탄1·2 신도시, 위례 신도시, 하남미사, 판교 신도시 |
| 적용 대상 (정책 제안) | 하남교산 신도시 |

## 2. 어떻게 분석을 하였는지

### 핵심 개념: EPDO 점수란?

EPDO(Equivalent Property Damage Only)는 **교통사고의 심각도를 숫자로 환산한 점수**입니다.
단순히 "몇 건 났다"가 아니라, **얼마나 심각한 사고였는지**를 반영합니다.

> **비유**: 골프에서 점수가 낮을수록 좋듯이, 교통사고에서 EPDO는 높을수록 위험합니다.

```
EPDO 가중치 (출처: 이상엽, 2019, 대한교통학회지)
┌──────────┬────────┐
│  사고 유형  │  점수  │
├──────────┼────────┤
│  사망      │  391점 │  ← 가장 심각
│  중상      │   69점 │
│  경상      │    8점 │
│  부상신고   │    6점 │
│  상해없음   │    1점 │
└──────────┴────────┘
```

예) 사망사고 1건 = 경상사고 약 49건과 동일한 위험도

---

### 분석 8단계 흐름도

```
[원시 데이터]
교통사고 이력 (10,821건) + 도로망 + 교통량 + 안전시설 + 인구 + 토지이용
        │
        ▼
[STEP 01] 사고별 EPDO 점수 부여
  각 사고에 심각도에 따른 점수를 붙인다
        │
        ▼
[STEP 02] 도로 링크별 교통량 집계
  도로 한 구간에 하루 몇 대가 지나가는지 계산 (노출량)
        │
        ▼
[STEP 03] 도로 링크별 위험도 산출
  위험도 = EPDO 합계 / 노출량 × 100만
  → 차량 100만 대·km당 EPDO 점수
        │
        ▼
[STEP 04] 고위험 도로 원인 분석
  위험도 상위 20개 도로의 사고 원인 분석
  (위반 유형, 사고 유형, 시간대, 요일 분포)
        │
        ▼
[STEP 05] 격자별 EPDO + 인프라 통합
  공간을 격자(바둑판 셀)로 나눠 각 셀의 위험도와
  안전시설(횡단보도·CCTV 등) 현황 통합
        │
        ▼
[STEP 06] 인프라 공백 분석
  위험도 높은 격자 중 안전시설이 부족한 구역 추출
  → 시설 설치 우선순위 도출
        │
        ▼
[STEP 07] 링크 위험도 보강 — 속도 + 혼잡강도
  도로별 평균속도·혼잡빈도·혼잡시간을 결합해
  보정된 위험도(enhanced_rate) 산출
        │
        ▼
[STEP 08] 격자 종합 위험 지수 산출 + 하남교산 예측
  교통약자 인구, 인프라 공백, 도로 속도·혼잡,
  취약 시간대 인구, 주말 집중도를 모두 반영한
  최종 위험 점수 산출 → 하남교산 신도시 예측 적용
```

## 3. 어떻게 예측을 하였는지

> ※ 이 분석에서 "예측"은 머신러닝이나 AI 모델을 쓴 것이 아니라,
> **기존 신도시 데이터의 패턴을 측정해 위험 지역을 사전에 식별**하는 방식입니다.

### 방법 1: 도로 위험도 점수(EPDO Rate)로 위험 도로 순위 매기기

단순히 "사고가 많이 난 도로"를 찾는 것은 불공평합니다.
교통량이 많은 도로는 당연히 사고도 많이 납니다.

그래서 **"차량 통행량 대비 사고가 얼마나 심각한가"** 를 따집니다:

```
위험도(EPDO Rate) = 전체 EPDO 점수 합계 ÷ 노출량 × 1,000,000

노출량 = 하루 평균 통행 차량 수(AADT) × 도로 길이(km)
```

**예시로 이해하기**:
- A 도로: 차량 많이 통과, EPDO 합 100점 → 위험도 낮음
- B 도로: 차량 거의 없음, EPDO 합 10점 → 위험도 훨씬 높음

→ B 도로가 더 위험합니다. 차량이 거의 안 다니는데 심각한 사고가 났기 때문.

---

### 방법 2: 속도·혼잡강도 보정 위험도 (STEP 07)

단순 EPDO Rate에 도로의 주행 특성을 더해 **보정 위험도(enhanced_rate)**를 산출합니다.

```
보정 위험도 = EPDO Rate × 속도 가중치 × 혼잡 가중치

속도 가중치 = 평균속도(km/h) ÷ 60
  → 60km/h 초과 시 보행자 치사율 급증 (교통안전공단 기준)
  → 빠를수록 위험도 상승

혼잡 위험도 = 혼잡빈도강도 × 혼잡시간강도 ÷ 100
  → 자주 막히고 심하게 막힐수록 위험도 상승
```

---

### 방법 3: 격자 단위 인프라 공백 분석으로 위험 구역 식별

도시 전체를 **바둑판처럼 격자(셀)로 나눠서**:

```
각 격자 셀 분석:
  ① 이 격자에서 발생한 사고의 EPDO 점수 합산
  ② 이 격자에 설치된 안전시설 수 확인
     (횡단보도, CCTV, 과속방지턱, 어린이보호구역 등)
  ③ 고위험(상위 25%)인데 시설이 부족한 격자 = 우선 설치 대상
```

**시설 공백 판단 기준**:
- 시설이 0개 → "없음"
- 전체 평균의 50% 미만 → "부족"

> **[개선] 공백 대상 시설 재정의**: 교육시설(학교·유치원·어린이집)은 "없어도 위험이 높아지지 않는 위험 유발 시설"이므로 공백 지표에서 제외.
> 횡단보도·어린이보호구역·과속방지턱·CCTV·버스정류장 **6개 안전시설**만을 기준으로 공백을 산출합니다.

### 방법 4: 격자 종합 위험 지수 — 엔트로피 가중치 적용 (STEP 08 + 09)

도로 위험도에 **교통약자 인구·속도·혼잡·시간대·주말 집중도**를 반영하되,
각 인자의 기여 비중을 **엔트로피 가중법**으로 객관적으로 산출해 적용합니다.

```
보정지수 = 각 인자를 0~1로 정규화 후 가중합

  46.23% × 노인 거주 비율
+ 35.98% × 취약시간대 인구 (등교·하교·퇴근)
+  6.68% × 혼잡 위험도
+  4.82% × 격자 평균속도
+  3.22% × 주말 방문 비율
+  2.39% × 인프라 공백 수
+  0.67% × 교통약자 유동 비율
─────────────────────────────────
= 보정지수 (0 ~ 1 사이 값)

종합 위험 지수 = EPDO 합계 × (1 + 보정지수)
```

> 기존에는 7개 인자를 모두 동등하게 곱했으나, 엔트로피 가중법 적용 후
> **노인 거주 비율(46.2%)과 취약시간대 인구(36.0%)** 두 인자가 전체의 82.2%를 차지해
> 격자 간 위험도 차별화에 가장 크게 기여함을 데이터로 확인했습니다.
> 또한 교육시설을 인프라 공백 지표에서 제외(9종→6종)한 결과 **인프라 공백 수의 가중치가 1.75%→2.39%로 상승**,
> 속도 데이터 커버리지 개선(58.4% → 92.5%)으로 혼잡(6.7%)·속도(4.8%) 인자의 가중치도 반영됐습니다.

### 방법 5: 상관관계 분석으로 어떤 시설이 효과적인지 확인

EPDO 점수와 각 안전시설 수 사이의 **상관관계(Pearson, Spearman)**를 계산해
"어떤 시설이 사고 심각도와 관련이 있는가"를 측정했습니다.

```
상관계수 해석:
  +0.1 이상 → 시설 많은 곳일수록 EPDO도 높음 (교통량 많은 지역일 가능성)
  -0.1 이하 → 시설 많은 곳일수록 EPDO 낮음 (억제 효과 가능)
  -0.1 ~ +0.1 → 뚜렷한 관계 없음
```

### 방법 6: 하남교산 신도시 예측

기존 4개 신도시의 분석 결과를 토지이용 유형별로 평균 내어,
하남교산의 각 계획 구역에 **동일 유형의 위험도 패턴을 전이**합니다.

```
예측 방식:
  ① 4개 신도시에서 "학교용지"의 평균 종합 위험 지수 계산
  ② 하남교산 토지이용 계획도에서 "학교용지" 구역에 해당 값 적용
  ③ 동일 유형 없으면 → 유사 카테고리 평균 → 전체 평균 순으로 대체
```

### STEP 01: `01_epdo_score.py` — 사고별 EPDO 점수 부여

**하는 일**: 교통사고 기록에 심각도별 가중치 점수를 붙이는 가장 기초 작업입니다.

```
입력: output/13._교통사고_링크매핑.csv (사고 이력 + 도로 링크 매핑)
출력: epdo_analysis/output/01_사고별_EPDO점수.csv
```

**핵심 로직**:
1. 각 사고 행(row)을 읽어 `injury_svrity`(사고 심각도) 컬럼을 확인
2. 사전에 정의된 가중치 표에서 해당 심각도의 점수를 조회
3. 점수를 `epdo_score` 컬럼으로 저장

```python
EPDO_WEIGHTS = {
    "사망":    391,   # 사망자가 발생한 사고
    "중상":     69,   # 3주 이상 입원이 필요한 부상
    "경상":      8,   # 5일 이상 치료 필요
    "부상신고":   6,   # 경미한 부상
    "상해없음":   1,   # 물적 피해만 존재
}
```

**유의사항**: 심각도가 기재되지 않은 421건은 0점 처리 (과소평가 방지를 위한 보수적 처리)


**핵심 로직**:
1. 도로망 파일에서 각 링크의 "상행 방향 ID"와 "하행 방향 ID"를 추출
2. 교통량 파일에서 방향별 AADT(연평균일교통량)를 읽어 링크에 연결
3. 상행+하행의 평균을 내어 링크별 총 교통량 산출
4. **노출량 = AADT × 도로 길이(km)** 계산 (단위: 대·km/일)

```
노출량 예시:
  AADT 10,000대 × 길이 0.5km = 5,000 대·km/일
  → 이 링크에서 하루 5,000번의 차량-km 노출이 발생
```

### STEP 03: `03_link_epdo_risk.py` — 도로 링크별 위험도 산출

**하는 일**: STEP 01의 EPDO 점수와 STEP 02의 교통량을 결합해
각 도로 구간의 **정규화된 위험도**를 계산합니다.

```
입력: 01_사고별_EPDO점수.csv + 02_링크별_교통량.csv
출력: epdo_analysis/output/03_링크별_위험도.csv
```

**핵심 공식**:
```
EPDO Rate = (해당 링크의 EPDO 합계) ÷ (노출량) × 1,000,000

단위: 백만 대·km당 EPDO 점수
```

**왜 100만을 곱하나?**: 노출량 단위가 커서 계산 결과가 너무 작은 숫자가 되는 것을 방지.
교통 공학에서 표준적으로 사용하는 정규화 방법입니다.

**결과물 구조**:
- 모든 링크에 `epdo_rank`(위험도 순위) 부여
- 교통량 데이터가 없는 링크는 `epdo_rate = null`로 처리

### STEP 06: `06_infra_analysis.py` — 인프라 공백 분석

**하는 일**: 고위험 격자 중에서 안전시설이 부족한 구역을 찾아냅니다.
또한 어떤 시설이 사고 심각도와 통계적으로 관련이 있는지 측정합니다.

```
입력: epdo_analysis/output/05_격자별_EPDO_인프라통합.csv
출력: epdo_analysis/output/06_인프라_상관분석.csv
      epdo_analysis/output/06_인프라공백_고위험격자.csv
```

**분석 1 — 상관관계 계산**:
- 외부 라이브러리(numpy, pandas 등) 없이 **순수 Python으로 직접 구현**
- Pearson 상관계수: 선형 관계 측정
- Spearman 상관계수: 순위 기반 비선형 관계 측정

```python
# Pearson 상관계수 직접 구현
def pearson(x, y):
    mx, my = mean(x), mean(y)
    분자 = Σ((x_i - 평균x) × (y_i - 평균y))
    분모 = √(Σ(x_i - 평균x)²) × √(Σ(y_i - 평균y)²)
    return 분자 / 분모
```

**분석 2 — 인프라 공백 격자 추출**:
1. EPDO 상위 25% 격자를 "고위험 격자"로 정의
2. 각 격자의 시설 수가 전체 평균의 50% 미만이면 "부족"
3. 0개이면 "없음"으로 표시
4. 공백 시설 수가 많은 순서로 정렬 → 우선 설치 대상 목록 생성

### STEP 07: `07_link_speed_congestion.py` — 속도·혼잡 보강 위험도

**하는 일**: 기존 EPDO Rate에 **도로별 주행 속도와 교통 혼잡도**를 결합해
더 실질적인 보정 위험도(enhanced_rate)를 산출합니다.

```
입력: 03_링크별_위험도.csv
      data/09._평균속도.csv     (v_link_id별 시간대별 속도)
      data/11._혼잡빈도강도.csv  (v_link_id별 FRIN_CG)
      data/12._혼잡시간강도.csv  (v_link_id별 TI_CG)
      data/08.상세도로망_네트워크.geojson (v_link_id → link_id 매핑)
출력: epdo_analysis/output/07_링크_속도혼잡_보강.csv
```

**핵심 공식**:
```python
# 속도 가중치: 60km/h 초과할수록 위험도 상승
speed_weight = avg_speed / 60.0

# 혼잡 위험도: 자주 막히고 오래 막힐수록 높음
congestion_risk = FRIN_CG × TI_CG / 100

# 보정 위험도 = EPDO Rate × 속도 가중치 × (1 + 혼잡 가중치)
enhanced_rate = epdo_rate × speed_weight × (1 + congestion_risk / 100)
```

**추가 분석**:
- 교통약자 취약 시간대(07~09시 등교, 16~19시 하교·퇴근)의 `peak_speed` 별도 산출
- 속도 구간(고속/준고속/중속/저속)별 라벨 자동 부여

### STEP 08: `08_grid_composite_risk.py` — 격자 종합 위험 지수

**하는 일**: 지금까지 분석한 모든 데이터를 하나의 점수로 통합합니다.
단순 사고 통계를 넘어 **"실제로 교통약자가 위험에 노출되는 격자"**를 정밀하게 찾아냅니다.
추가로 하남교산 신도시의 예측 위험도도 산출합니다.

```
입력: 06_인프라공백_고위험격자.csv, 07_링크_속도혼잡_보강.csv
      data/01 격자, data/03~07 인구, data/08 도로망,
      data/22 토지이용(4개 신도시), data/23 토지이용(하남교산)
출력: epdo_analysis/output/08_격자_종합위험지수.csv
      epdo_analysis/output/08_하남교산_예측위험도.csv
```

**종합 위험 지수 공식 (엔트로피 가중치 적용)**:
```python
# STEP 09 CSV에서 동적 로드 (하드코딩 제거)
# 최신 가중치 (v3: 안전시설 6종 기준 적용)
ENTROPY_W = {
    "elderly_res_ratio": 0.4623,   # 노인 거주 비율     (46.2%)
    "vuln_peak_pop":     0.3598,   # 취약시간대 인구    (36.0%)
    "grid_congestion":   0.0668,   # 혼잡 위험도        (6.7%)
    "grid_avg_speed":    0.0482,   # 격자 평균속도       (4.8%)
    "weekend_ratio":     0.0322,   # 주말 방문 비율      (3.2%)
    "gap_cnt":           0.0239,   # 인프라 공백 수      (2.4%) ← 교육시설 제외 후 상승
    "vuln_float_ratio":  0.0067,   # 교통약자 유동 비율  (0.7%)
}

# 각 인자를 0~1로 정규화 후 가중합
correction_index = Σ(ENTROPY_W[인자] × 정규화값)

# 최종 종합 위험 지수
entropy_composite_risk = epdo_total × (1 + correction_index)
```

**기존 공식과의 차이**:

| 구분 | 기존 (composite_risk) | 신규 (entropy_composite_risk) |
|------|--------------|------------------|
| 방식 | EPDO × 5개 보정계수 단순 곱셈 | EPDO × (1 + 교통약자 중심 보정지수) |
| 인자 비중 | infra_penalty·speed_weight가 지배적 | 노인거주(46%)·취약시간인구(36%) 중심 |
| 1위 격자 | 다사672369 (3,454점) | 다사672368 (851.3점) |
| 근거 | 없음 (임의) | 엔트로피 가중법 (객관적) |

> ⚠️ **두 순위(composite_rank vs entropy_rank)의 최대 차이는 1,132위** 이며 이는 정상입니다.
> 두 지표는 서로 다른 위험 요소를 측정합니다:
> - `composite_rank`: **시설 공백·속도** 중심 — 인프라 전체 없음(infra_penalty=2.0)이면 EPDO가 낮아도 높은 순위
> - `entropy_rank`: **교통약자 인구 집중** 중심 — 노인거주·취약시간 인구가 많은 격자를 우선
>
> 정책 집행 시 **최종 우선순위 = entropy_rank**를 기준으로 삼고,
> entropy_rank 상위이면서 gap_cnt가 높은 격자를 "인프라 우선 설치 대상"으로 교차 선정할 것을 권장합니다.

**기술적 특이사항**:
- `geopandas`의 공간 조인(sjoin)을 활용해 경위도 인구 데이터를 격자에 매핑
- **[개선] 도로 선분 intersects 기반 공간 조인**: 기존 링크 중심점(centroid) 방식에서 링크 선분이 격자와 교차하는 방식으로 변경 → 속도·혼잡 커버리지 58.4% → **92.5%**로 향상
- **[개선] 도로 등급별 보간**: 속도 미계측 링크가 통과하는 격자에 동일 도로 등급의 평균 속도·혼잡도를 적용해 결측 최소화
- **[개선] 엔트로피 가중치 동적 로드**: STEP 09 CSV를 런타임에 읽어 자동 적용 (fallback 상수 보유)
- **[개선] 교통약자유동多 라벨 임계값**: 고정값(0.15) → 전체 분포 75%ile(~0.34) 동적 적용 → 99.2% → 24.9%로 정상화
- **[개선] 인프라 공백 패널티**: 교육시설 제외 6개 안전시설 기준, 분모 9 → 6으로 수정
- 좌표계 변환: WGS84(EPSG:4326) → 한국 중부원점(EPSG:5186)으로 재투영 후 공간 연산 수행
- 하남교산 예측: 동일 토지이용 유형 → 유사 카테고리 → 전체 평균 순으로 위험도 전이
