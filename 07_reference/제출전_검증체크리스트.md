# Submission Readiness Checklist

## Code execution
- [ ] `requirements.txt` exists
- [ ] `README_run.md` exists
- [ ] All scripts run without errors in clean environment
- [ ] Notebook files (`*.ipynb`) are included

## Encoding
- [ ] Python files are UTF-8
- [ ] CSV outputs saved as UTF-8 (recommended `utf-8-sig`)

## QGIS files
- [ ] QGIS project file included (`.qgz` or `.qgs`)
- [ ] Style files included (`.qml`)
- [ ] Layer paths are relative
- [ ] Project opens without broken layers

## CRS
- [ ] All submitted SHP/GeoJSON are EPSG:4326
- [ ] QGIS project CRS is EPSG:4326
- [ ] `python 07_reference/check_crs_epsg4326.py` passes

## Final deliverables
- [ ] Data ZIP includes preprocessed/analysis/visualization outputs
- [ ] Analysis report exists in PPT/PDF
- [ ] Presentation script file exists
- [ ] Fonts used in report are attached
